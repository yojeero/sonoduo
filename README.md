### [Arumo](https://arumo.vercel.app/) is a online radio player. ###

```
Features
```

- Switchable themes
- 5 online radio stations presents
- Any number of radio stations & covers in vue.player.js
- Vue.js based player
- Clock with time and day of week
- 100% Responsive Design
- Flexbox
- SVG icons 
- Google Fonts
- jQuery, VueJS
- Animations
